Git-開新專案 and 基本使用

Posted on 2022-11-01 Edited on 2022-11-16

解釋

Git ：分散式的版本控制系統
集中式：集中上傳到同一地點，從同一地點下載
分散式：本地端可指定給任一個本地端
集中式缺點：集中的端點壞掉就全部不能用了。
Git 同時支援本地與遠端操作。
